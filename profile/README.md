# 🚀 Labora Backend

Microservice-based backend for a Freelancing Platform built using Django and Django REST Framework.

---

## 🧠 Architecture

This project follows a **microservice architecture** where each service is independently developed and managed.

---

## ⚙️ Services

- 📩 Message Service → Real-time chat & notifications  
- 💼 Job Service → Job posting & applications  
- 💳 Payment Service → Transaction & payment tracking  
- 🛠 Admin Service → Platform control & monitoring  
- 👤 Client Profile Service → Client data management  
- 👨‍💻 Freelancer Profile Service → Freelancer profile & skills  

---

## 🛠 Tech Stack

- Django & Django REST Framework  
- JWT Authentication  
- WebSocket (Django Channels)  
- MySQL  

---

## 📌 Highlights

- Role-based system (Client, Freelancer, Admin)  
- Real-time communication  
- Modular scalable backend  
- REST API based architecture
