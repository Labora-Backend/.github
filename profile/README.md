# Labora Backend

Microservice-based backend for a Freelancing Platform.

## Architecture
This project follows a microservice architecture where each service is independently developed and managed.

## Services

- Message Service  
- Job Service  
- Payment Service  
- Admin Service  
- Client Profile Service  
- Freelancer Profile Service  

## Tech Stack
- Django & Django REST Framework  
- JWT Authentication  
- WebSocket (Django Channels)  
- MySQL
